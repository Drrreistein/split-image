## 🔧 高级配置

### 修改端口
编辑 `app.py` 文件：
```python
app.run(debug=True, host='0.0.0.0', port=9000)  # 修改端口号
```

### 修改文件大小限制
编辑 `app.py` 文件：
```python
app.config['MAX_CONTENT_LENGTH'] = 50 * 1024 * 1024  # 50MB
```

### 添加更多图片格式
编辑 `app.py` 文件：
```python
app.config['ALLOWED_EXTENSIONS'] = {'png', 'jpg', 'jpeg', 'webp', 'bmp', 'gif'}  # 添加 gif
```
