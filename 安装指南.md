# 长图切分 Skill 安装指南

## 📁 文件说明

本项目包含以下文件：

```
/Users/a1-6/WorkBuddy/20260312221420/
├── split_image.py              # 核心切分脚本
├── split-image-skill/          # Skill 完整包（可直接安装）
│   ├── skill.md               # Skill 配置文件
│   ├── split_image.py         # 脚本副本
│   └── README.md              # 详细使用文档
├── 使用说明.md                # 中文使用指南
└── test_script.py             # 测试脚本（可选）
```

## 🚀 安装方法

### 方法1：作为 Skill 安装（推荐）

将 `split-image-skill` 文件夹复制到 WorkBuddy 的 skills 目录：

```bash
# macOS/Linux
cp -r split-image-skill ~/.workbuddy/skills/

# Windows
xcopy /E /I split-image-skill %USERPROFILE%\.workbuddy\skills\split-image-skill
```

安装后，在对话中直接说：
```
帮我把这张长图切分成5份
图片路径：/path/to/image.png
```

### 方法2：直接使用脚本

```bash
# 安装依赖
pip install Pillow

# 运行脚本
python split_image.py <图片路径> <切分数量>
```

## 📖 使用示例

### 示例1：基本使用
```bash
python split_image.py screenshot.png 5
```
输出：`screenshot_切分/part_01.png` ~ `part_05.png`

### 示例2：指定输出目录
```bash
python split_image.py article.jpg 3 ./output
```
输出：`./output/part_01.png` ~ `part_03.png`

### 示例3：作为 Skill 使用
```
用户：帮我把 /Users/xxx/Desktop/long.png 切成4份
助手：[执行切分] → 已完成，输出目录：/Users/xxx/Desktop/long_切分
```

## 🔧 依赖要求

- Python 3.6+
- Pillow (PIL)

安装依赖：
```bash
pip install Pillow
```

## 📝 功能特点

✅ 支持多种图片格式（PNG、JPG、JPEG、WEBP等）
✅ 自动计算均匀切分高度
✅ 自动创建输出文件夹
✅ 保留原图质量（PNG无损输出）
✅ 支持自定义输出路径
✅ 简单易用的命令行接口
✅ 可作为 WorkBuddy Skill 使用

## ⚠️ 注意事项

1. 切分数量必须 ≥ 1
2. 适用于纵向长图（高度 > 宽度）
3. 建议使用绝对路径
4. 输出统一为 PNG 格式

## 🐛 常见问题

**Q: 提示找不到 Pillow 模块？**
```bash
pip install Pillow
```

**Q: 如何验证是否安装成功？**
运行测试脚本：`python test_script.py`

**Q: 切分后图片质量如何？**
无损切分，输出为 PNG 格式，保留原图质量

**Q: 可以切分横向长图吗？**
当前版本仅支持纵向切分，横向长图需要先旋转90度
