## 🚀 快速开始

### 1. 安装依赖

```bash
pip install Pillow
```

### 2. 使用脚本

**基本用法：**
```bash
python split_image.py <图片路径> <切分数量>
```

**示例：**
```bash
# 将长图切分成5份
python split_image.py screenshot.png 5

# 将长图切分成3份，保存到指定目录
python split_image.py long_image.jpg 3 ./output
```

## 📖 功能说明

### 核心功能
- 将长图按垂直方向均匀切分
- 自动计算每段高度
- 自动创建输出文件夹
- 支持常见图片格式

### 输入参数
- `图片路径`：要切分的图片文件路径
- `切分数量`：切分成几张图片（正整数）
- `输出目录`：（可选）切分后图片保存目录

### 输出结果
- 默认输出目录：`{原图片名}_切分/`
- 文件命名：`part_01.png`, `part_02.png`, ...
- 格式：PNG（无损）

## ⚙️ 技术细节

### 切分算法
- 总高度 ÷ 切分数量 = 每段高度
- 最后一张图片包含剩余像素（避免丢失内容）
- 按垂直方向从上到下切分

### 图片处理
- 使用 Pillow (PIL) 库
- 无损切分，保留原图质量
- 统一输出 PNG 格式

## 📋 注意事项

1. **切分数量**：必须 ≥ 1
2. **图片方向**：适用于纵向长图（高度 > 宽度）
3. **路径建议**：使用绝对路径避免问题
4. **输出格式**：统一为 PNG，保证质量
