# 快速部署到 Render

## 🚀 一键部署到 Render

### 步骤1：访问 Render

访问：https://render.com

### 步骤2：连接 GitHub

1. 点击 "Get Started for Free"
2. 选择 "Sign up with GitHub"
3. 授权 Render 访问你的 GitHub 账户

### 步骤3：创建新的 Web Service

1. 进入 Dashboard
2. 点击 "New +" 按钮
3. 选择 "Web Service"
4. 选择 "Build and deploy from a Git repository"
5. 点击 "Next"

### 步骤4：连接仓库

1. 点击 "Connect account"
2. 找到并选择 `Drrreistein/split-image`
3. 点击 "Connect"

### 步骤5：配置服务

填写以下信息：

- **Name**: `image-splitter`（或你喜欢的名字）
- **Region**: 选择离你最近的区域（如 Singapore）
- **Branch**: `main`
- **Root Directory**: `web_app` ⚠️ 重要！
- **Runtime**: `Python 3`
- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `python app.py`
- **Instance Type**: `Free`

### 步骤6：高级设置（可选）

点击 "Advanced" 添加环境变量：

- `FLASK_DEBUG` = `False`
- `SECRET_KEY` = `<随机字符串>`

### 步骤7：部署

1. 点击 "Create Web Service"
2. 等待部署完成（约2-3分钟）
3. 部署成功后会显示绿色的 "Live" 状态

### 步骤8：访问应用

点击应用名称或URL，格式类似：
```
https://image-splitter-xxxx.onrender.com
```

## 📊 Render 免费版限制

- **运行时间**: 每月 750 小时（足够24/7运行）
- **内存**: 512MB
- **CPU**: 共享CPU
- **存储**: 无持久化存储（重启后文件会丢失）
- **带宽**: 100GB/月

## ⚠️ 注意事项

### 关于文件存储

Render 免费版没有持久化存储，意味着：
- 应用重启后，上传的文件会丢失
- 建议：处理完立即下载

### 解决方案

1. **立即下载**: 切分完成后立即下载
2. **使用持久化存储**: 升级到 Render 付费版
3. **集成云存储**: 使用 AWS S3、Cloudflare R2 等

## 🔄 自动部署

配置完成后，每次推送到 `main` 分支，Render 会自动重新部署：

```bash
git add .
git commit -m "Update app"
git push
```

## 🎉 完成！

你的应用现在已经在云端运行了！
