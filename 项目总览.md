## 🚀 快速开始

### 1. 命令行版本


## 📐 核心算法

### 均匀切分算法
```python
# 计算每段高度
part_height = total_height // num_parts

# 切分逻辑
for i in range(num_parts):
    if i == num_parts - 1:
        # 最后一张包含剩余像素
        height = total_height - y_offset
    else:
        height = part_height
    
    # 裁剪并保存
    crop_and_save(y_offset, height)
    y_offset += height
```

## 🐛 常见问题

### Q1: 如何选择使用哪个版本？
- **日常使用**：Web版（可视化、易操作）
- **批量处理**：命令行版（高效、可编程）
- **智能助手**：Skill版（自然语言交互）

### Q2: 文件大小限制可以调整吗？
可以，编辑 `web_app/app.py`：
```python
app.config['MAX_CONTENT_LENGTH'] = 100 * 1024 * 1024  # 改为100MB
```

### Q3: 支持横向长图吗？
当前版本仅支持纵向切分。横向长图需要先旋转90度。

### Q4: 如何批量处理多张图片？
使用命令行版本配合shell脚本：
```bash
for img in *.png; do
    python split_image.py "$img" 5
done
```
