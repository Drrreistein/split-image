# 长图均匀切分工具

一个简单易用的长图切分工具，可以将一张长图按照指定数量均匀切分成多张图片。

## 功能特点

- ✅ 支持多种图片格式（PNG、JPG、JPEG、WEBP等）
- ✅ 自动计算均匀切分高度
- ✅ 自动创建输出文件夹
- ✅ 保留原图质量
- ✅ 支持自定义输出路径
- ✅ 简单易用的命令行接口

## 安装依赖

```bash
pip install Pillow
```

## 使用方法

### 基本用法

```bash
python split_image.py <图片路径> <切分数量>
```

### 指定输出目录

```bash
python split_image.py <图片路径> <切分数量> <输出目录>
```

## 使用示例

### 示例1：切分长截图

```bash
python split_image.py screenshot.png 5
```

这将在 `screenshot_切分/` 文件夹中生成5张切分后的图片：
- part_01.png
- part_02.png
- part_03.png
- part_04.png
- part_05.png

### 示例2：指定输出目录

```bash
python split_image.py long_image.jpg 3 ./my_output
```

这将在 `./my_output/` 文件夹中生成3张切分后的图片。

## 参数说明

| 参数 | 说明 | 是否必需 |
|------|------|---------|
| 图片路径 | 要切分的图片文件路径 | 必需 |
| 切分数量 | 切分成几张图片 | 必需 |
| 输出目录 | 切分后图片保存目录 | 可选 |

## 输出说明

- **默认输出目录**：`{原图片名}_切分/`
- **文件命名格式**：`part_01.png`, `part_02.png`, ...
- **图片格式**：统一输出为 PNG 格式，保证质量

## 注意事项

1. 切分数量必须大于0
2. 如果切分数量不能整除图片高度，最后一张图片会包含剩余像素
3. 建议使用绝对路径以避免路径问题
4. 切分按垂直方向进行，适用于纵向长图

## 技术细节

- 使用 Python Pillow 库处理图片
- 采用均匀切分算法，自动计算每段高度
- 支持大图片处理，内存优化
- 保留原始图片质量，无压缩损失
