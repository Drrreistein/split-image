# 长图均匀切分 Skill

## 描述
这是一个用于将长图按照指定数量均匀切分的工具技能。适用于需要将长截图、长文章图片等内容分割成多个独立图片的场景。

## 功能特点
- 支持任意常见图片格式（PNG、JPG、JPEG、WEBP等）
- 自动计算均匀切分高度
- 自动创建输出文件夹
- 保留原图质量
- 支持自定义输出路径

## 使用场景
当用户需要：
- 将长截图切分成多个部分
- 将长文章图片分割便于阅读
- 将聊天记录长图分段保存
- 处理需要分页显示的长图

## 工作流程

### 参数获取
1. 询问用户图片路径（如果未提供）
2. 询问切分数量（如果未提供）
3. 询问是否指定输出目录（可选，默认在原图同目录创建）

### 执行切分
调用 `split_image.py` 脚本执行切分操作：
```python
python split_image.py <图片路径> <切分数量> [输出目录]
```

### 结果反馈
告知用户切分完成情况，包括：
- 生成的图片数量
- 每张图片的尺寸
- 输出目录位置

## 依赖要求
- Python 3.6+
- Pillow 库（PIL）

安装依赖：
```bash
pip install Pillow
```

## 示例用法

**示例1：基本用法**
```
用户：帮我把这个长截图切分成5份
助手：请提供图片路径
用户：/Users/example/Desktop/screenshot.png
助手：[执行切分] → 输出结果
```

**示例2：指定输出目录**
```
用户：把这个长图切成3份，保存到 ./output 文件夹
助手：[执行切分] → 输出结果
```

## 脚本位置
脚本文件：`split_image.py`

## 输出格式
- 文件命名：part_01.png, part_02.png, ..., part_N.png
- 输出目录：原图文件名_切分/
